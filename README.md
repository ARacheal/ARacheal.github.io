# Antonita Racheal — Personal Website

The source for [aracheal.github.io](https://aracheal.github.io) — a personal portfolio for Antonita Racheal, data analyst at the Metropolitan Transportation Authority.

## Built with

- HTML, CSS, and vanilla JavaScript
- Hosted on GitHub Pages
- Inline SVG hero (Brooklyn Bridge), glassmorphism cards, dark mode, and a parallax scroll

## Local preview

Open `index.html` in a browser, or serve the folder:

```bash
python -m http.server 8000
```

Then visit `http://localhost:8000`.

## Structure

- `index.html` — page markup
- `styles.css` — theme variables, layout, and components
- `script.js` — dark mode, scroll reveal, parallax, card tilt
- `images/` — photos and thumbnails
